### dev'ing:

To run this mug:

- `mix setup`
- `bin/server`

For dev mode: [`localhost:4002`](http://localhost:4002)

For fly mode: [`megalithic.fly.dev`](https://megalithic.fly.dev)

For live mode: [`megalithic.io`](https://megalithic.io)
